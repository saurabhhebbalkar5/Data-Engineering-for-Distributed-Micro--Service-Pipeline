1. Change the file path to yellow_tripdata_2018-01 in the Sortfile.py
	This will generate the SortedTaxiData.csv in the current folder, which will be used to publish later

2. Run the SortFile.py

3. Place taxi+_zone_lookup.csv and Motor_Vehicle_Collisions_-_Crashes.csv in the current working directory

4. Load the google.guava jar into the library(any IDE)

5. Run YellowTaxi.java file.
	This will initiate both publisher and subscriber.
